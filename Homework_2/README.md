# Homework 2. Django маршрутизация

## Задание 1
Используя механизм маршрутизации создано веб-приложение, отображающее разные разделы
с информацией о городе.

Реализованы разделы:
- Главная
- Новости города
- Руководство города
- Факты о городе
- Контактные телефоны городских служб

Каждый раздел доступен по своему адресу:
- 127.0.0.1:8000 — Главная
- 127.0.0.1:8000/news — Новости города
- 127.0.0.1:8000/management — Руководство города
- 127.0.0.1:8000/facts — Факты о городе
- 127.0.0.1:8000/contacts — Контактные телефоны

---

## Задание 2
Добавлена обработка неправильных адресов.

При вводе некорректных путей (например:
`/news/satisfaction`, `/news/angie`, `/news/wildhorses`)
отображается содержимое соответствующего основного раздела.

Аналогичный принцип реализован для всех разделов приложения.

---

## Задание 3
Добавлен раздел **История**.

Реализованы следующие маршруты:
- 127.0.0.1:8000/history — общая информация об истории города
- 127.0.0.1:8000/history/people — известные жители города
- 127.0.0.1:8000/history/photos — исторические фотографии города
